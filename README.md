## 功能

- 允许无条件使用第三方主题。
- 允许免费使用所有上架的字体。
- 去除应用内部的广告，不包括开屏广告(因为开屏广告来自另一个 MIUI 应用[智能服务])。
- 使用 `Apktool` 在模块刷入时自动解包修改你当前安装的版本。
- 支持 MIUI11 至 MIUI13 中后期所有的主题壁纸 App。
