## 功能

- 允许无条件使用第三方主题。
- 允许免费使用所有上架的字体。
- 去除应用内部的广告,不包括开屏广告(因为开屏广告来自另一个MIUI应用)。
- 使用`Apktool`在模块刷入时自动解包修改你当前安装的版本。
- 截至目前支持MIUI11及之后所有的主题壁纸版本。
